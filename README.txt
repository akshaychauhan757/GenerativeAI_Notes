\## 🚀 Project Overview

\*\*SynthAnalytics\*\* is an enterprise-grade framework designed to combat data scarcity in Machine Learning. By utilizing \*\*Conditional Tabular GANs (CTGAN)\*\*, the system learns the underlying statistical DNA of a small dataset and synthesizes high-fidelity, privacy-safe data at scale.



\## 🛠️ Tech Stack

\* \*\*Engine:\*\* Python 3.14 + PyTorch

\* \*\*Generative Model:\*\* CTGAN (Conditional Tabular Generative Adversarial Network)

\* \*\*Frontend:\*\* Streamlit (Custom Glassmorphic UI)

\* \*\*Orchestration:\*\* Modular Python Architecture



\## 📂 Project Structure

\* `app.py`: The Luxury UI and Frontend Logic.

\* `src/engine.py`: The Deep Learning Backend \& GAN Training Protocol.

\* `requirements.txt`: System Dependency Registry.



\## ⚡ Quick Start

1\. \*\*Activate Environment:\*\* `synth\_env\\Scripts\\activate`

2\. \*\*Install Dependencies:\*\* `pip install -r requirements.txt`

3\. \*\*Launch Engine:\*\* `streamlit run app.py`
